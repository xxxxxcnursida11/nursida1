Program OutPut

Explanation:

The HTML <samp> element represents outputform a program or computing system. Text surrounded by <samp> tags is typically displayed in the browser default mono-space font:

Sample code:

<!DOCTYPE html>
<html>
    <body>
        <h2> The samp Element </h2>
        <p> The samp element is used to define sample output form a computer program. </p>
        <p> <samp> File not found. <br> Press F1 to continue</samp> </p>
    </body>
</html>
