computer code element

Explanation:

The HTML <code> element defines a fragment of computer code. Text surrounded by <code> tags is typically displayed in the browser default mono-space font:

Sample code:

<!DOCTYPE html>
<html>
    <body>
        <h2> The code Element </h2>
        <p> Programming code example </p>
<code>
x = 5
y = 6
z = x + y;
</code>
    </body>
</html>

OUTPUT:

The code Element

Programming code example

x = 5 y = 6 z = x + y;
